

Description: Executable files used to run the calibration/verification model
	           and all scenarios. These executables have been run on the 
	           64-bit Windows 7, Windows Server 2008 HPC, and Windows 
	           Server 2012 R2 Standard Operating Systems.
	           
	           The executables were compiled using the Intel(R) Visual Fortran 
	           64-bit compiler for Windows (version 14.0.0092.11) and Visual
	           C++ 2012 compiler.


Contents:

  o MD_mfnwt_x64.exe - MODFLOW-NWT executable used for calibration run 
                      (UMD.final) and all scenario runs (UMD.final.scenarios)
                      64-bit executable
  o ZoneBud_x64.exe - Modified version of ZoneBudget that was used to calculate
                      budget for select zones
                      64-bit executable
